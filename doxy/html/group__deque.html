Detailed Description

a double-ended queue data structure.

a double-ended queue is an abstract data type that generalizes a queue, for which elements can be added to or removed from either the front (head) or back (tail). It is also often called a head-tail linked list, though properly this refers to a specific data structure implementation of a deque.

Format Specifier

  • %deq(#) - # is a non-empty character or string specifying the format specifier of that type contained by the deque
  • {deq(#)} - prints the deque to the stdout.
