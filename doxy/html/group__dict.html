Detailed Description

A hash table (hash map) is a data structure used to implement an associative array.

a hash table (hash map) is a data structure used to implement an associative array, a structure that can map keys to values. A hash table uses a hash function to compute an index into an array of buckets or slots, from which the desired value can be found.

Format Specifier

  • %dict(#:#) - # is a non-empty character or string specifying the format specifier of that type contained by the dict
  • {dict(#:#)} - prints the dict to the stdout.
  • %[#:#] - # is a non-empty character or string specifying the format specifier of that type contained by the dict
  • {[#:#]} - prints the dict to the stdout.

Example

Creating a dict.

ads_dict d = ads_new(ads_dict,int,ads_hashInt,int,32);
ads_dict d1 = ads_dictNew(ads_str,ads_strHash,int);

inserting some key-value pairs into the dict.

ads_dictSetItem(d,5,int,5*10,int);

printing the dict.

ads_printf("dict = {[d:d]} ",d);

checks weather a dict has a particular key.

ads_dictHasKey(d,10,int); //return true if the dict has the key 10 else false

fetch a values through a key.

ads_dictGetItem_Copy(d,5,int,int,val); //returns the value corresponding to the key

delete a dict.

ads_delete(d,d1); //returns the value corresponding to the key
